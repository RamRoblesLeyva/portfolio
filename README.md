## 🔹 Azure SQL Automated Backup System

### Descripción

Sistema automatizado para generar respaldos de bases de datos SQL por tabla, generando scripts `.sql` que permiten restaurar datos fácilmente.

### Tecnologías

- Azure Automation
- PowerShell
- Azure SQL
- Microsoft Graph API
- OneDrive

### Funcionalidades

- Generación automática de scripts `CREATE TABLE`
- Generación de `INSERT` por tabla
- Backup automático diario
- Creación de carpetas por fecha
- Envío automático de respaldo por correo

### Impacto

- Eliminación de procesos manuales de respaldo
- Mejora en recuperación ante incidentes
- Automatización completa del proceso
